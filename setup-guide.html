📚 英単語チャレンジ セットアップガイド

このガイドに従えば、無料でウェブアプリを公開できます（約30分）。

1 Googleアカウントで Firebase にアクセス

https://console.firebase.google.com を開き、「プロジェクトを追加」をクリック。

プロジェクト名（例：eigo-challenge）を入力し、Google アナリティクスは「無効」でOKです。

2 Firestore データベースを作成

左メニューの 「Firestore Database」 → 「データベースの作成」をクリック。

ロケーションは asia-northeast1（東京） を選択してください。

「本番モードで開始」を選んで作成します。

3 アプリの設定コードを取得

プロジェクトの 「設定（歯車アイコン）」→「プロジェクトの設定」→「マイアプリ」 から 「ウェブアプリを追加（</>）」 をクリック。

以下のような設定コードが表示されます：

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "eigo-challenge.firebaseapp.com",
  projectId: "eigo-challenge",
  storageBucket: "eigo-challenge.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
};

この内容を index.html の該当部分（YOUR_API_KEY などの箇所）に貼り付けてください。

4 セキュリティルールを設定

Firestore の 「ルール」タブ を開き、以下のルールを貼り付けて「公開」をクリック：

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /students/{studentId} {
      allow read, write: if true;
    }
    match /words/{wordId} {
      allow read: if true;
      allow write: if true;
    }
  }
}

⚠️ これは簡易ルールです。誰でも書き込める状態になっています。本格運用時はFirebase Authenticationを導入することをお勧めします。

5 Firebase Hosting でサイトを公開

以下の2つの方法があります：

方法A：Firebase Hosting（推奨・無料）

  • Node.js をインストール（nodejs.org）
  • ターミナルで以下を実行：
npm install -g firebase-tools
firebase login
firebase init hosting
firebase deploy

firebase init hosting では：

  • 「Use an existing project」→ 先ほど作ったプロジェクトを選択
  • 「What do you want to use as your public directory?」→ . と入力（Enterで確定）
  • 「Configure as a single-page app?」→ N
  • 「Set up automatic builds?」→ N

完了すると https://eigo-challenge.web.app のようなURLが発行されます！

方法B：Netlify Drop（最も簡単・無料）

  • https://app.netlify.com/drop を開く
  • 「英単語練習アプリ」フォルダをブラウザにドラッグ＆ドロップするだけ！
  • すぐにURLが発行されます

6 パスワードの変更

app.js を開き、先頭の CONFIG を編集してください：

const CONFIG = {
  studentPassword: 'eigo2024',   // ← 生徒に教えるパスワード
  teacherPassword: 'sensei2024', // ← 先生だけが知るパスワード
  ...

変更後、再度サイトをデプロイ（アップロード）してください。

✅ 完成！使い方まとめ

  • 🎓 生徒：URLにアクセス → 生徒パスワードで入る → 名前・学年・クラスを登録 → 練習スタート
  • 👩‍🏫 先生：URLにアクセス → 「先生」タブ → 先生パスワードで入る → 単語を登録
  • 📝 単語登録：先生ダッシュボードで学年・クラス・レッスンを選び、英語と日本語を入力
  • 🏆 ランク：ペンギン(0pt〜) → イルカ(100pt〜) → パンダ(300pt〜) → とら(600pt〜) → ユニコーン(1000pt〜)
